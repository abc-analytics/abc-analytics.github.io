  • What is this so-called .NET framework for layman. It is basically the Microsoft development ecosystem that replicates (or duplicates) the Java environment (intermediate language compilation, library, data access interface, etc); while Java is OS-portable, .NET is practically a Windows thing. Hence for a Java developer, the .NET is what he has been used to for years and is more vendor-specific.
