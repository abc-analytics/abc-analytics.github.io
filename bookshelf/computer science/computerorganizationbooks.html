  • Code
  • Charles Petzold
  • It might not seem like it, but it is a solid book on computer architecture from binary numbers and Boolean logic to electrical circuits and memory chips in showing how information can be transformed/represented/encoded into binary/electrical/digital forms. By providing background story and following a logical progression, it is more reader-friendly than standard computer organization texts where switches/gates, arithmetics, and architectures are introduced. Through examples and graphs, he describes how Boolean algebra can be used to perform abstract logical tests and then how electric circuits can be wired to implement Boolean logic in practice. He describes how binary numbers can be added/subtracted by combining OR, AND, NAND, XOR gates into Adders; he then describes how gates can be combined into flip-flops to retain bits and act as dynamic memory circuits. After touching upon ALU and CPU basics, he concludes by a brief discussion of high-level programming languages (which are converted into low-level manchine codes to interact with processors) and modern GUI-based OS.
