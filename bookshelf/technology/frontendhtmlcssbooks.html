  • It introduces JS in order to discuss the new HTML APIs (geolocation, video, JSONP/XMLHttprequest, local storage, thread workers). The materials like web services are rather complex and difficult to fully cover (cross-domain security etc), but it still manages to get the key ideas through, using easy-to-imagine examples of sales-update (for data exchange) and sticky-note (for local storage). Canvas and worker chapters skipped. Should dig into each major API by a detailed book or online documentation, but the guided examples here are good as exercises.
